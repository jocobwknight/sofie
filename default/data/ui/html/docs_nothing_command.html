SoFiE Documentation

nothing

Description

The nothing command does, as you might suspect, much less than something. It is a placeholder and template. But if you want to run it and you're not paying the electric bill, then knock yourself out!

Syntax

nothing fake_argument=<string> [pretend_bool=<true|false>]

Required arguments

fake_argument
Syntax: fake_argument=<string>
Description: This isn't actually a required argument. There are no required arguments. How could "nothing" require "something?"

Optional arguments

pretend_bool
Syntax: pretend_bool=<true|false|t|f>
Description: You could put this in if you want. You could also NOT put it in. In either case, this command will do: nothing

Basic Examples

Example 1:

Do nothing

... | nothing

Extended Examples

Example 2:

Do nothing with twice as much style

index=*
| head 1
| nothing
